# Dona ChatBot

A simple console-based chatbot built with Python.

## Features

- Responds to greetings
- Answers simple questions
- Shows available commands with `help`
- Handles unknown messages
- Exit command
- Easy to read and modify

## Commands

- hello
- hi
- how are you
- your name
- creator
- thank you
- help
- bye
- exit

## How to Run

1. Make sure Python is installed.
2. Download the project.
3. Open a terminal in the project folder.
4. Run:

```bash
python chatbot.py
```

## Example

```
You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: creator
Bot: I was created by Dona :)
```
